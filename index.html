sistema dinámico

El sistema dinámico es ideal para operaciones que requieren mantener una operación de inventario basada en Primeras Entradas Primeras Salidas (PEPS), su diseño permite gestionar y movilizar grandes flujos de mercancía de manera fácil y eficiente.

Los elementos básicos de nuestros sistemas dinámicos son las cabeceras, largueros y el rodamiento, cada uno de ellos forma parte esencial de la solución diseñada para cada almacén, las cabeceras son fabricadas de acero estructural o formado, son unidas por los largueros con los cuales se genera la estructura que recibirá las líneas de rodamiento que constituyen el sistema dinámico.

Las líneas de rodamiento son fabricadas en rodillo galvanizado y/o negro dependiendo del entorno del almacén y/o necesidades del cliente.

El sistema opera ingresando producto/mercancía por la parte posterior del módulo realizado, esta parte es las más alta del sistema, lo cual permite el correcto deslizamiento hacia el punto de salida gracias a la gravedad y configuración del rodamiento establecido.

Ventajas:
Un solo punto de carga y descarga de producto.
Control de inventario UEPS
Aprovechamiento de paredes.
Alta densidad en espacios reducidos
Aumenta hasta un 80% la capacidad de almacenamiento vs un sistema selectivo.
Idóneo para el almacenamiento de productos de mediana rotación.

Push Back

Sistema ideal para espacios limitados, permite mantener una operación de inventario de Ultimas Entradas Ultima Salidas (UEPS), su diseño permite almacenar hasta 8 tarimas de fondo sin necesidad de pasillos y utiliza un solo punto de entrada y salida, toma ventaja de la gravedad para el desplazamiento de las tarimas.

Al igual que los sistemas dinámicos , los sistemas Push Back tienen como elementos básicos de las cabeceras, largueros y el rodamiento, cada uno de ellos forma parte esencial de la solución diseñada para cada almacén, las cabeceras son fabricadas de acero estructural o formado, son unidas por los largueros con los cuales se genera la estructura que recibirá las líneas de rodamiento que constituyen el sistema Push Back, el rodamiento puede ser con rodillo o rodajas, dependiendo del tipo de tarimas a utilizar.

El sistema opera ingresando producto/mercancía por la parte frontal del módulo, esta parte es las más baja del sistema, lo cual permite el correcto deslizamiento hacia el punto de salida gracias a la gravedad y configuración del rodamiento establecido.

Ventajas:
Un solo punto de carga y descarga de producto.
Control de inventario UEPS
Aprovechamiento de paredes.
Alta densidad en espacios reducidos
Aumenta hasta un 80% la capacidad de almacenamiento vs un sistema selectivo.
Idóneo para el almacenamiento de productos de mediana rotación.

Picking

Uno de los sistemas más completos debido a su versatilidad en el manejo de tarimas, cajas y piezas sueltas, todas ellas concentradas en un mismo punto de manera organizada y eficiente, el aumento de productividad es una de las grandes ventajas de este sistema.

La integración de sistemas selectivo, dinámico, carton flow y push back proporciona toda la variabilidad en un solo sistema, contiene cabeceras estructurales o formadas, rodillos galvanizados o negros, rodamiento de rodaja o ruedas de aluminio, miniruedas para carton flow, todos ellos pueden convivir en el mismo sistema.

El sistema funciona para la preparación de pedidos extrayendo unidades o paquetes de cada una de las posiciones, el sistema genera un pasillo central que permite a los operadores seleccionar los productos establecidos en su hoja de producto, de igual forma el sistema es reabastecido de las partes superiores para permitir una constante operación de preparación de pedidos.

Ventajas:
Alta diversidad de productos
Gran volumen de armado de pedidos
Disminución en el armado de órdenes.
Uso de sistemas superiores para almacenamiento.
Gestión de productos Tipo “C”.
Optimización de espacios.
Reducción de recorridos.

Carton Flow

Diseñado especialmente para el manejo de una alta diversidad de productos en un mismo punto, especial para manejo de cajas y/o paquetes, su uso principal es en área de armado de pedidos o picking como comúnmente se conoce.

El sistema utiliza ruedas o rodillos de menor tamaño que permiten el desplazamiento de las cajas y/o paquetes, montado sobre rieles y soportado bajo una estructura de selectivo el sistema permite crear niveles y frentes para el desplazamiento del producto de un punto más alto al punto más bajo.

El sistema es alimentado por la parte posterior y utilizando la gravedad como fuerza motor permite deslizarse hacia el punto más bajo de la pendiente establecida y necesaria por la operación.

Ventajas:
Manejo de PEPS.
Cero consumos de energía eléctrica.
Mayor cantidad de productos en un área pequeña.
Menor tiempo en armado de pedidos.

Double Deep

Ideal para aumentar la capacidad de almacenamiento a dos tarimas de fondo utilizando el principio de UEPS, se genera una alta selectividad de productos y disminución de pasillos, para el uso de un sistema de doble fondo (Double Deep) es necesario utilizar un montacargas de doble alcance.

El sistema consta de cabeceras estructurales o formadas con entrepaños o parrillas según sea la necesidad las cuales permiten una mejor estabilidad del producto a almacenar.

La carga y descarga de los bienes es por el mismo punto, el sistema debe posicionarse frente a la ubicación donde se realizará la operación, deberá ingresar primeramente la tarima del fondo y posteriormente la frontal.

Ventajas:
Manejo de alta selectividad.
Aumento de hasta un 50% de la capacidad de almacenamiento.

Selectivo

Este sistema proporciona acceso directo a las tarimas las cuales pueden ser tomadas individualmente y de forma rápida, están diseñados para el manejo de cualquier número de productos y cualquier nivel de rotación.

El sistema está compuesto por acero estructural o formado dependiendo de la necesidad del cliente, lo conforman cabeceras, diagonales, travesaños, largueros y/o entrepaños o parrillas de carga, pudiendo soportar tarimas de hasta 1500 kilos.

La operación es simple, solo es necesario colocar el producto en la posición asignada, este puede ser en cualquiera de los niveles del sistema.

Ventajas:
Gran selectividad
Rápido acceso a los productos.
Aumento de almacenaje

Drive In

Este sistema proporciona acceso directo a las tarimas las cuales pueden ser tomadas individualmente y de manera rápida, la naturaleza de su diseño está enfocada al manejo de cualquier número de productos y de cualquier nivel de rotación.

Compuesto por cabeceras, diagonales y rieles permite crear un módulo los suficientemente compacto y abierto para el tránsito de montacargas por el interior del mismo.

Al generar los pasillos de transito es posible ingresar con el producto para ser colocado en los niveles del sistema.

Ventajas:
Configuración del sistema a cualquier espacio, tamaño y peso de la mercancía.
Alto volumen de almacenamiento.
Disminución de pasillo al máximo.

cantiliever

Diseñado especialmente para materiales de gran longitud o dimensiones, como podrían ser tuberías, rejillas o paneles, pueden ser sencillo o dobles.

Su composición es sumamente básica, consta de columnas y brazo que permiten depositar el producto en el nivel deseado.

El producto puede ser depositado en montacargas o por personal de la empresa dependiendo de las características del producto.

Ventajas:
Es fácil de instalar y utilizar.
Estructura configurable.
Ahorro de espacio.

Autosoportados

Los sistemas autosoportados son una excelente opción para nuevos almacenes, este tipo de sistema permiten utilizar la estructura de los racks para soportar la estructura del edificio, logrando con ello reducir considerablemente los costos de ingeniería civil.

El sistema utiliza las cabeceras como parte central del mismo, para crear la estructura se emplean perfiles de acero que soportan el recubrimiento de lámina o panel según sea el caso, este tipo de sistemas es ideal para cuartos fríos y congeladores.

Este tipo de almacén puede ser utilizando con sistemas dinámicos, selectivos, semiautomáticos, automáticos y drive in, en base a la necesidad la operación se puede adecuar a los sistemas ya mencionados.

Características principales:
Máximo espacio disponible a un bajo costo.
Rapidez en su construcción vs un almacén tradicional.
Sistema para grandes alturas.
Ideal para ambientes fríos o de congelación.

Automatización

Mexroll ofrece soluciones para el manejo de materiales, ello incluye sistemas automáticos de tarimas para sus almacenes y plantas productivas, cada proyecto es analizado y evaluado en conjunto con los usuarios y dueños del proyecto hasta llegar a una solución idónea.

El incremento de velocidad, disminución de accidentes y la productividad son algunas de las principales ventajas de los sistemas automáticos y proyectos desarrollados por el equipo de automatización.

Permiten trasladar los productos o materias primas a lo largo de una trayectoria establecida, logran disminuir los tiempos de entrega hacia la zona de almacén o embarque y líneas productivas.

Larguiero Tipo Caja

De ensamble rápido y fácil con el sistema "ojo de llave" el cual une el larguero a la cabecera sin necesidad de atornillar, además de contar con un dispositivo de seguridad integrado para evitar el desensamble accidental de los largueros.


Carga Pesada

Gran capacidad de carga para optimizar el uso de las áreas de almacenaje. Enorme flexibilidad para modificar su arreglo. Diseñados para ensamblarse en minutos con un mínimo de trabajo (sistema ojo de llave). Fácil y económico, con posibilidad para extender su capacidad mediante la incorporación.


Larguero tipo escalon

Este larguero permite el ensamble de accesorios de carga tales como entrepaños, parrillas, etc.


Brazos Cantiliever

Larguero Estructural
